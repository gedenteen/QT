Введение

Данная программа предназначена для ведения личного дневника. Однако вы можете использовать ее по своему усмотрению. Основной функционал программы:

  • Создание записей.
  • Возможность выбрать дату, для любой даты можно сделать любое количество записей.
  • Поиск по записям в выбранном дне.

Оглавление

  1. Календарь
  2. Создание записей
  3. Поиск по записям
  4. Кнопки
  5. Настройки

Календарь

Календарь находится слева-вверху. На календаре отображается выбранный месяц, учитывая дни недели. Кликнув по нужной дате, вы выбираете этот день. Месяц и год можно менять с помощью кнопок над календарем. Также дни, в которых были сделаны записи, отмечены желтым задним фоном.

Создание записей

После выбора нужного дня, вы можете создать новую запись - кнопка для этого всегда находится в самом низу справа. Если в этот день уже были сделаны записи, то вы можете посмотреть или отредактировать их.

Запись состоит из заголовка, текста, хэштегов. Текст записи форматируется с помощью HTML

Поиск по записям

Для поиска достаточно ввести нужное слово или слова (и нажать кнопку "Поиск". Если есть совпадение, то будут выведены нужные записи. Поиск ведется по заголовку, тексту и тегам.

Кнопки

  • "Удалить запись" - безвозвратно удаляет выбранную запись.
  • "Сохранить запись" - сохранить новую запись или сохранить изменения в уже созданной записи.
  • "Форматирование текста" - применить форматирование HTML к записи. Если форматирование уже применино к записи, то показывает текст с HTML-тегами.
  • "Шаблоны" - использовать шаблон для записи, который содержит некоторую "заготовочную" информацию. Внимание! Если изменения не были сохранены, то они будут безвозвратно удалить.
  • "Добавить" - добавить определенные теги к концу текста записи.

Настройки

Файл

  • "Открыть" - открыть файл с базой данных.
  • "Сохранить" - сохранить изменения, если есть.
  • "Сохранить как" - сохранить копию текущей базы данных в другом файле.
  • "Выгрузить в txt-файл" - выгрузить все записи в txt-файл (записи можно просмотреть через текстовый редактор).
  • "Выход" - выйти из приложения.
